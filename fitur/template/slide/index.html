Cara Membuat Slide Sederhana Dengan HTML, CSS, dan Javascript

Oleh : Tristan Ahnaf W.

19 Agustus 2023

Halo teman-teman, pada kesempatan kali ini kita akan membuat sebuah slide sederhana menggunakan bahasa pemprograman HTML, CSS, dan Javascript.

Slide ini sendiri dapat teman-teman kembangkan atau teman-teman gunakan sebagai slide presentasi, perkenalan diri, halaman website, preview halaman, dan lain-lain. Sehingga dapat menjadi slide yang lebih baik lagi dan multifungsi.

Sebenarnya ada banyak cara untuk membuat sebuah slide dengan Bahasa pemprograman, namun pada kesempatan kali ini kita akan menggunakan HTML, CSS, dan Javascript.

Untuk project kali ini teman-teman juga tidak perlu menggunakan library atau extension dari luar, karena semua element yang kita gunakan akan menggunakan element dari HTML nya langsung.

HTML

Yang pertama kita buat sebuah project HTML dengan diberi nama ‘index.html’ Untuk aplikasi dan cara membuat file HTML sudah saya jelaskan di artikel berikut (klik disini). Selanjutnya teman-teman dapat mengcopy teks html dan Javascript yang sudah saya buat dibawah ini ke dalam project teman-teman.

Jika susah teman-teman dapat membaca petunjuk yang sudah saya tulis di dalam kotak untuk Langkah selanjutnya.

CSS

Selanjutnya kita tambahkan style CSS agar slide yang sudah kita buat tadi menjadi lebih menarik.

Caranya teman-teman dapat mengcopy style CSS yang sudah saya siapkan dibawah ini ke halaman HTML yang sudah kalian buat. Untuk bagaimana cara membuat file CSS teman-teman dapat membaca artikel saya yang ini (klik disini).

Javascript

Selanjutnya kita tambahkan Javascript agar Slide yang sudah kita buat tadi dapat berfungsi.

Caranya teman-teman dapat mengcopy Javascript yang sudah saya siapkan dibawah ini ke halaman HTML pada bagian yang sudah ditandai. Untuk bagaimana cara membuat file JS teman-teman dapat membaca artikel saya yang ini (klik disini).
